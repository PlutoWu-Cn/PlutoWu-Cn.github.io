修改Java字节码

在OpenJDK里有一个AsmTools项目，用来生成正确的或者不正确的java .class文件，主要用来测试和验证。

AsmTools引入了两种表示.class文件的语法：

  • JASM

    用类似java本身的语法来定义类和函数，字节码指令则很像传统的汇编。

  • JCOD

    整个.class用容器的方式来表示，可以很清楚表示类文件的结构。

重要的是两种语法的文件都是可以和.class互相转换的。

在这里主要使用JASM语法。
